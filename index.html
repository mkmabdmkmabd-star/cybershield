// 01 — Project Overview

What is
CyberShield?

CyberShield is an embedded wireless security monitoring system built on the ESP32-WROOM-32 microcontroller. It operates as a standalone defense node capable of scanning the 2.4GHz spectrum, classifying threats, and alerting in real-time.

The system runs in AP+STA dual mode, hosting its own Wi-Fi access point at 192.168.4.1 while simultaneously performing passive and active scans of the RF environment. A built-in buzzer on GPIO 25 produces distinct audio signatures for each threat class.

A live cybersecurity dashboard is served over HTTP — accessible from any browser on the same network — displaying detected networks, attack counters, signal analysis, and a rolling event log.

// 02 — Threat Detection Engine

What We Detect

⚡
Deauth Attack
Detects IEEE 802.11 deauthentication frame floods — a classic disconnect-injection attack that forces clients off legitimate access points.
CRITICAL 2× beep · 800Hz
🌊
DDoS / Flood
Identifies abnormal traffic density patterns indicative of probe request floods, association floods, or management frame storms targeting local infrastructure.
HIGH 1× beep · 1000Hz
📡
RF Jammer
Detects active RF jamming via two heuristics: sudden disappearance of previously-known networks, and detection of anomalously high-power signal sources (RSSI > −30 dBm).
ACTIVE Continuous · 2000Hz
👁
Hidden Networks
Passive detection of networks broadcasting with empty or null SSIDs — a common technique used by rogue access points and covert infrastructure.
PASSIVE Logged · Flagged
🍍
Rogue AP / Evil Twin
Identifies access points broadcasting with suspiciously high signal strength at close range, characteristic of portable attack hardware like Wi-Fi Pineapple devices.
WARNING Suspicious Flag
🔔
Beacon Flood
Detects abnormal proliferation of unique SSIDs across channels, a pattern consistent with beacon flooding tools designed to overwhelm Wi-Fi scanning utilities.
MEDIUM Heuristic Match

// 05 — Hardware

Bill of Materials

01
ESP32-WROOM-32
Dual-core 240MHz Xtensa LX6 · 520KB SRAM · 4MB Flash · Built-in 2.4GHz Wi-Fi + BT. The brain of the entire system.
02
Active Buzzer
Connected to GPIO 25. Generates distinct alert tones — 800Hz for Deauth, 1kHz for DDoS, 2kHz continuous for active jammers.
03
TP4056 LiPo Charger
Li-ion/LiPo charging module with protection circuit. Enables portable battery-powered operation without mains dependency.
04
Push Button
Manual trigger for system control — reset counters, force scan, or silence alarm without accessing the web interface.
